# Índice de Ruteo de Deliverables — Estructura de Proyecto

Documento derivado del catálogo SDLC enriquecido para soportar generación de estructura de proyecto, setup de sprints y handoffs.

## Columnas

| Columna | Qué contiene |
|---------|--------------|
| **Responsable** | Rol dueño del resultado final |
| **Ejecuta** | Rol o agente que hace el trabajo operativo |
| **Spec Source** | El archivo markdown individual del deliverable, ya separado y enriquecido |
| **Sección** | `completo`, porque cada deliverable ya vive en un archivo dedicado |
| **Horas** | Horas derivadas desde `typical_effort` o `TBD` si requiere decisión manual |
| **Complejidad** | `LOW`, `MEDIUM`, `HIGH` o `TBD` si requiere decisión manual |
| **Dependencias** | IDs de deliverables predecesores |
| **Docs para el agente** | Documento fuente del diccionario + template asociado |
