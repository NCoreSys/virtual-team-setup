## Leyenda

| Símbolo | Significado |
|---------|-------------|
| `→` (línea sólida) | Actor inicia el caso de uso |
| `..> <<include>>` | UC siempre ejecuta otro UC como sub-flujo |
| `..> <<extend>>` | UC condicionalmente ejecuta otro UC |
| `🤖` | Actor sistema (automático) |
| `👤` | Actor humano |
| `⏰` | Actor interno (cron/timer) |
