## DoR — Checklist Universal

Una tarea está "Ready" para moverse a `task_in_progress` cuando cumple TODOS estos criterios:

### Documentación

- [ ] ASSIGNMENT leído completamente (no solo el BRIEF)
- [ ] Todas las dependencias están en `task_completed` en VTT
- [ ] Ambigüedades identificadas en el ASSIGNMENT están resueltas o escaladas al SA Reviewer
- [ ] Fuentes de referencia listadas en el ASSIGNMENT son accesibles (archivos existen, URLs funcionan)

### Entorno

- [ ] BD `memory_service_db` accesible desde el entorno de desarrollo
- [ ] Storage `/storage/` montado y con permisos de escritura
- [ ] Variables de entorno configuradas: `DATABASE_URL`, `SERVICE_KEY`, `REDIS_URL` (si aplica)
- [ ] Rama Git creada: `feature/MS-XXX`
- [ ] Dependencias npm instaladas (`npm install` sin errores)

### Contexto técnico

- [ ] SPEC v1.9 disponible y leída (al menos las secciones relevantes)
- [ ] Documentos de análisis relevantes disponibles (RFs, UCs, BRs, ACs de la feature)
- [ ] Si la tarea modifica schema: migration de Prisma verificada en entorno local
- [ ] Si la tarea depende de catálogos: seed ejecutado (`prisma db seed`)

## DoR — Criterios Adicionales por Tipo de Tarea

### Tareas de Backend (endpoints)

- [ ] Contrato del endpoint definido en SPEC §8 (request/response)
- [ ] Validaciones Zod definidas en 2.5.3
- [ ] Business Rules aplicables identificadas en 2.5.2
- [ ] Acceptance Criteria Gherkin disponibles en 2.7.2

### Tareas de Frontend (componentes)

- [ ] Design specs del DL disponibles (wireframes, componentes, tokens)
- [ ] Endpoints backend que consume están implementados y probados
- [ ] Datos de prueba disponibles en BD local

### Tareas de Testing (QA)

- [ ] Feature a probar está deployada en entorno de testing
- [ ] Test scenarios definidos en 2.7.5
- [ ] Datos de prueba preparados (conversaciones importadas, catálogos seeded)

### Tareas de Documentación (SA/AR)

- [ ] Documentos fuente listados en el ASSIGNMENT están disponibles
- [ ] Tareas previas de las que depende están aprobadas (no solo completadas)
