## TS-01 · Idempotencia de Import

| Campo | Valor |
|-------|-------|
| **Feature** | Import (EP-01) |
| **US** | US-001, US-002, US-003 |
| **BRs** | BR-001 |
| **Prioridad** | Crítica |

| # | Escenario | Input | Acción | Resultado esperado |
|---|-----------|-------|--------|-------------------|
| 1 | Import primera vez | JSONL válido, externalSessionId nuevo | POST /import | HTTP 201, status: IMPORTED |
| 2 | Reimport mismo ID | Mismo externalSessionId | POST /import | HTTP 200, status: ALREADY_INDEXED |
| 3 | Reimport sin cambios en BD | Verificar BD post-reimport | COUNT conversations | Mismo count que antes del reimport |
| 4 | Reimport sin cambios en storage | Verificar /storage/ post-reimport | ls -la archivo | Mismo timestamp/size |
| 5 | Race condition | 2 requests simultáneos, mismo ID | POST /import × 2 concurrentes | Uno IMPORTED, otro ALREADY_INDEXED, 0 duplicados |
