## POST /api/conversations/import

| Campo | Tipo | Requerido | Validación | Ref. |
|-------|------|-----------|------------|------|
| `sourceCode` | string | Sí | Enum: `CLAUDE_SDK`, `CLAUDE_CLI`, `CLAUDE_WEB`, `CHATGPT` | D-MEM-09 |
| `externalSessionId` | string | Sí | No vacío. Para CLAUDE_SDK: formato `{run_id}:r{N}:{agentRole}` | D-INT-02 |
| `jsonlContent` | string | Sí | JSONL válido (parseable línea por línea) | D-MEM-18 |
| `projectId` | string | No | UUID válido si presente | — |
| `taskId` | string | No | UUID válido si presente | D-MEM-11 |
| `taskKey` | string | No | Max 20 chars | D-MEM-11 |
| `taskTitle` | string | No | Max 200 chars | D-MEM-12 |
| `primaryAgentId` | string | No | UUID válido si presente. Requerido para TASK_EXECUTION | D-MEM-36 |
| `agentRole` | string | No | Max 50 chars | D-MEM-38 |
| `description` | string | No | Max 2000 chars | — |
| `platformRefs` | object | No | JSON object. Para CLAUDE_SDK: debe incluir `runtime_run_id` (string) | D-INT-03, D-INT-04 |

**Error:** HTTP 400 con detalle Zod si falla validación.
